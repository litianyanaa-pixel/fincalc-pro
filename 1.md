# 二、核心功能模块

------

# 1. 收益率计算模块

用户输入：

- 买入金额
- 持仓收益
- 持仓金额
- 持有时长

自动计算：

- 收益率
- 年化收益率

------

## 智能联动逻辑

### 持仓收益 ↔ 持仓金额双向自动计算

用户填写：

### 情况1：

填写：

- 买入金额
- 持仓收益

自动计算：

- 持仓金额

公式：

持仓金额=买入金额+持仓收益

------

### 情况2：

填写：

- 买入金额
- 持仓金额

自动计算：

- 持仓收益

公式：

持仓收益=持仓金额-买入金额

------

# 2. 收益反推模块（独立 Tab）

用户输入：

- 年化收益率
- 本金金额
- 持有时长

自动计算：

- 到期收益
- 到期总金额

公式：

收益=本金\times 年化收益率\times 时间

------

# 3. 复利模式（独立 Tab）

- 单利

用户输入：
